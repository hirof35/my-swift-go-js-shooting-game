# Go × HTML5/Swift 弾幕シューティングゲーム

Go言語の高性能な並列処理（Goroutine）をバックエンドに採用し、WebSocket通信を通じてリアルタイムに敵の生成や弾幕パターン（通常・発狂）を制御するクロスプラットフォーム対応のシューティングゲームです。

## 🚀 特徴
- **ハイパフォーマンスなバックエンド**: Go言語によるミリ秒単位でのゲームロジック計算
- **リアルタイム双方向通信**: WebSocketによる超低遅延データ同期
- **マルチフロントエンド対応**: 
  - ブラウザですぐに遊べる **HTML5 / Canvas / JavaScript 版**
  - iOSネイティブで滑らかに動作する **Swift / SpriteKit 版**
- **動的なゲーム性**: ボスの残りHPに連動した2段階の弾幕変化（発狂モード搭載）
